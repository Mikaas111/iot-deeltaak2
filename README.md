Dit is de code voor het Windesheim HBO-ICT semester IOT deeltaak 2 van Max en Mika.

Het betreft een slimme, geluid makende, vogelverschrikker die werkt met een ESP32, en gebruik maakt van BLE-Mesh.
Detectie wordt gedaan met een pingsensor, en geluid wordt afgespeeld dmv een DAC met daarop aangesloten een amplifier en een speaker.
